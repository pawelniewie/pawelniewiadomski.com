Mac software that I most often use

25 Mar 2013

Here’s a list of Mac software that helps me with day to day duties.

Things - because I want to get things done

Evernote - because I want to remember things and organize them

Dropbox - because I want to share files between multiple devices

Writer - because I love the clarity when writing

Alfred 2 - because of the workflows

SourceTree - because I commit to git

Pixelmator - because I’m a visual creature

Day One - because I want to remember my life

Cobook - because that’s a social address book

HandBrake - because I want to watch movies

VLC - because I want to watch movies

RapidWeaver - because sometimes I want to make a web site

ScreenFlow - because I want to cut a video

Transmission - because I use torrent

TweetDeck - because I love multiple columns

You Need a Budget - because I’m a money-wise guy

F.lux - because I want to get some sleep
