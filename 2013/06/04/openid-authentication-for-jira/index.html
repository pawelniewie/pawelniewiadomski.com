OpenID Authentication for JIRA

04 Jun 2013

Here’s my new upcoming plugin for Atlassian JIRA that will let you to set up JIRA to accept OpenID authentication.

It will match users by email, and if the user doesn’t exist and JIRA is in public mode and uses internal user directory it will automatically create an account for the user and log him in.

No further configuration needed. You just install it, provide the license, and you’ll get nice buttons on the Log in page.

Currently supports Google and Yahoo. Soon to follow. Especially the custom OpenID server will be available next.
